# Redact

![Kotlin](https://img.shields.io/badge/kotlin-2.2.10-blue.svg?logo=kotlin)
![Jetpack Compose](https://img.shields.io/badge/Jetpack%20Compose-Ready-brightgreen.svg)

Лучший новостной агрегатор 2026 по версии разработчика.

## Возможности
* **Умная сводка новостей:** суммаризация новостей с помощью ИИ.
* **Интеграция Telegram и RSS-лент:** можно не тратить время на чтение рекламы и перепечаток новостей в десятках разношёрстных источников.
* **Пользовательский API ключ и выбор модели LLM:** сила и независимость.
* **Автоматическое обновление:** лента обновляется сама в заданное пользователем время.
* **Работа оффлайн:** уже собранные статьи можно читать без интернета. Картинки не прогрузятся, да и чёрт с ними - у вас останется текст.

## Технологии
* **Язык программирования:** Kotlin
* **UI-фреймворк:** Jetpack Compose
* **Сеть:** OkHttp
* **Интеграция LLM:** Google Gemini API
